# Renamer
Функция для генерации строки со ссылками на фотографии в формате, необходимом для автозагрузки товаров на Авито.

## Назначение
Функция создает готовую строку с путями к фотографиям товара, соответствующую требованиям платформы Авито для автоматической загрузки изображений.

### Требования к структуре файлов
Фотографии должны храниться в папках на Яндекс.Диске

Название папки соответствует идентификатору товара

Внутри каждой папки фотографии должны быть названы в формате:

1-1.jpg
1-2.jpg
1-3.jpg
и т.д.

#### Использование

`rename("название_папки_товара")`

##### Примечание

Функция генерирует пути для 8 фотографий (от 1-1.jpg до 1-8.jpg), разделенных символом |. Если в папке меньше фотографий, система Авито автоматически обработает только существующие файлы.
