# Renamer

Утилита для формирования строки с путями к фотографиям в формате, необходимом для автозагрузки товаров на Авито.

## Описание
Функция генерирует строку с путями к изображениям в папке на Яндекс.Диске. Строка содержит до 8 путей, разделённых символом `|`, в формате, ожидаемом системой автозагрузки Авито.

## Требования
- Фотографии находятся в папке на Яндекс.Диске.
- Название папки соответствует идентификатору товара.
- Имена файлов в папке должны иметь формат:
  - `1-1.jpg`
  - `1-2.jpg`
  - `1-3.jpg`
  - ...
  - `1-8.jpg`

Если в папке меньше файлов — строка включает 8 позиций, но Авито обработает только существующие файлы.

## Установка и запуск
Проект построен как модуль JavaScript (Vite-проект). Примеры команд (в корне проекта):
```bash
npm install
npm run dev
```

## Использование
В коде используется функция `rename(folderName)` для генерации строки и `events()` для привязки переключателей типа (товары / услуги).

Пример:
```js
import { rename } from "./src/renamer.js";

const result = rename("название_папки_товара");
console.log(result);
// yandex_disk://ПневмоСПБ-Товары/название_папки_товара/1-1.jpg | ... | /1-8.jpg
```

## API
- rename(elem: string): string  
  Возвращает строку с 8 путями к изображениям для переданной папки.

- events(): void  
  Привязывает обработчики к радиокнопкам для переключения базовой части пути:
  - Товары: `yandex_disk://ПневмоСПБ-Товары/`
  - Услуги:  `yandex_disk://ПневмоСПБ-Услуги/`

## Примечания
- Функция формирует пути на основе статической структуры и не проверяет существование файлов на диске.
- При использовании в браузере элементы формы в `src/main.js` показывают результат в реальном времени при вводе имени папки.
